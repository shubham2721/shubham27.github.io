Quora Question Pairs

The Quora Question Pairs ML project aims to detect whether two questions are semantically similar or duplicates. Using a dataset of question pairs, the project employs natural language processing techniques and machine learning models like Logistic Regression, SVM, and Decision Tree. Features such as question frequency, token counts, and text similarity are extracted. The models are evaluated using metrics like Log loss to predict if pairs of questions have similar meanings efficiently.
